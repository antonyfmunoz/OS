## How to Use This Registry

1. When creating a new tool skill, check here first

3. If NOT listed, WebSearch for official docs and add the tool here after research
4. The Speed column determines the staleness threshold (see update_intelligence.md)
5. Keep this file updated — add new tools as they're researched
