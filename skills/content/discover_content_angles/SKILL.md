# Skill: Discover Content Angles

## Purpose
Find content angles that will resonate with the ICP based on live intelligence signals.

## Outcome
3-5 ranked content angles with core tension, ICP resonance reasoning, hook direction, and content pillar alignment.

## Decision Criteria
- Run before any new content batch
- Run when hook performance is declining
- Run when Content Agent has no fresh intelligence
- Do not run without first reading Intelligence Agent output

## Execution Steps
1. Read latest ICP intelligence output
2. Identify top pain or aspiration the ICP is expressing right now
3. Identify the gap — what is NOT being said that the ICP needs to hear?
4. Generate 3-5 angles ranked by:
   - ICP resonance (how specific to them)
   - Originality (not saturated)
   - Brand fit (Vigilante Architect worldview)
5. For each angle: name it, state the tension, suggest the hook direction

## Failure Modes
- Generating from trends without ICP intelligence
- Generic angles that could be any creator's
- More than 5 angles (forces prioritization)
- Skipping the originality check

## Measurement
- Which angles become best-performing hooks (track via hook_performance_analysis)
- ICP reply rate to DMs using angle-derived hooks
