TemplateRegistry._load_from_path(path):
  - Opens path, reads each line as JSON, deserializes to TemplateCandidate
  - Returns dict[str, TemplateCandidate] keyed by template_id
  - Called for both template_candidates.jsonl (_candidates) and templates.jsonl (_promoted)

TemplateRegistry.find_matching(action_type, description) -> list[TemplateCandidate]:
  1. tpl_type = _infer_template_type(action_type, description)
  2. Matches _promoted dict by template_type == tpl_type AND status == PROMOTED
  3. Also checks _candidates where status in (APPROVED, CANDIDATE) AND confidence >= 0.5
  4. Returns sorted list: promoted first, then by confidence descending

_infer_template_type action_type mappings (from template_registry.py):
  "run_contradiction_engine" → CONTRADICTION_FIX
  "check_readiness" → READINESS_IMPROVEMENT
  "assess_state" → OBSERVATION_ACCURACY_FIX
  "execute" → MAINTENANCE_ACTION
  "create_api_route" → API_CONTRACT_FIX
  "run_probes" → MAINTENANCE_ACTION  (fallback)
  "identify_panel" → COCKPIT_PANEL_FIX
  "wire_bridge" → ROUTE_EXTRACTION_FIX
  "execute_maintenance" → MAINTENANCE_ACTION

_infer_template_type description fallbacks:
  "contradiction" in desc → CONTRADICTION_FIX
  "readiness" in desc → READINESS_IMPROVEMENT
  "deploy" in desc → DEPLOYMENT_VALIDATION
  "panel" or "cockpit" in desc → COCKPIT_PANEL_FIX
  "world model" or "observation" in desc → WORLD_MODEL_ACCURACY_FIX
  "test" in desc → TEST_REPAIR
  "route" or "endpoint" in desc → ENDPOINT_CONTRACT_FIX
  "dependency" in desc → DEPENDENCY_GRAPH_FIX
  fallback → MAINTENANCE_ACTION

AutonomousLanePolicy.require_template = True (default)
