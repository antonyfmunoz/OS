## Physical Moat Layers (Strategic Context Amendment v2)

**Physical Infrastructure Moat Doctrine**: Beyond software and services, the empire architecture includes physical infrastructure that becomes increasingly defensible:

| Layer | Description | When It Activates |
|-------|-------------|-------------------|
| Owned manufacturing | In-house production for proprietary products, custom parts, and portfolio companies | After vertical integration |
| Fulfillment infrastructure | Owned warehousing, packaging, shipping, and logistics for physical products | After first physical product revenue |
| Robotics automation | Automated manufacturing, assembly, quality control, and fulfillment | After manufacturing established |
| CNC/prototyping capability | In-house rapid prototyping, CNC machining, 3D printing for product development | Before full manufacturing — enables iteration |
| Custom components/actuators | Proprietary actuators, motors, sensors, grippers, controllers, and mechanical assemblies | Only where ownership creates moat, cost advantage, or supply-chain control |
| Physical infrastructure companies | Real estate, facilities, vertical integration assets owned under Munoz Holdings | 5-10+ year horizon |

**Anti-Copycat Moat Doctrine**: In an AI-native economy, pure software features are easily cloned. Physical infrastructure, manufacturing IP, supply chain ownership, operating history, fulfillment networks, and proprietary components are not. The combination of digital intelligence (UMH/EOS) with physical infrastructure creates a moat that neither software-only nor manufacturing-only competitors can replicate. See also: Physical Infrastructure Moat Doctrine, Physical Product Intelligence Layer.

**Custom Actuator / Robotics Component Doctrine**:
1. Buy standard actuators/components
2. Build products/machines using off-the-shelf parts
3. Learn failure modes, tolerances, costs, bottlenecks
4. Modify/customize strategic components
5. Prototype custom actuator assemblies
6. Manufacture selected high-leverage components
7. Build proprietary actuator/robotics systems only where they create moat

Core rule: Buy commodity. Customize strategic. Own bottleneck. Manufacture moat-critical.
