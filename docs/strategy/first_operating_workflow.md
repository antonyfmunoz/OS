# First Operating Workflow

**Workflow Name**: Personal Brand → Initiate Arena Revenue Loop
**Date**: 2026-05-03
**Status**: Current — this is the binding constraint until $10K/month net is stable

---

## Why This Workflow First

This is the shortest path from where the user is (pre-revenue, building infrastructure) to where the user needs to be (revenue, proven product, operating data). Every other ambition depends on this loop working.

This workflow touches all four active entities:
- **Personal Brand** = attention / distribution
- **Lyfe Institute** = product / revenue
- **Empyrean Studio** = creative / marketing / execution
- **OST / UMH / EOS** = operating system and substrate
