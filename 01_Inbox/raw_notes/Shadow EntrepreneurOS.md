# The Agents You Should Build First

Your system should start with **3 agents.**

---

# 1️⃣ Outreach Agent

Purpose:

Find leads and draft conversations.

Workflow:

1 scan target accounts  
2 identify commenters  
3 analyze profiles  
4 draft DM  
5 log conversation

Telegram command:

/find_leads

Output:

10 potential leads  
DM scripts  
profile notes

---

# 2️⃣ Content Agent

Purpose:

Turn your thoughts into content.

Example commands:

/content ideas  
/write script  
/viral hooks

Workflow:

1 analyze niche  
2 generate hooks  
3 write script  
4 create caption

---

# 3️⃣ Research Agent

Purpose:

Improve messaging.

Example:

/analyze objections  
/find comments  
/summarize audience pain

This helps refine your offer messaging.
