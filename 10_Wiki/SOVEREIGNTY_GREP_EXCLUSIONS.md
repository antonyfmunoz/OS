# Sovereignty Grep Exclusions

Canonical exclusion list for `scripts/sovereignty-grep.sh`. Each
excluded path has a documented rationale. Add new exclusions here
before adding them to the script.

See also: [[LAYER_3.1_SOVEREIGNTY_CLEANUP]] for the vocabulary
library and full retrospective.

source is clean, they're clean after regeneration. Stripping them
manually would be overwritten on next rebuild.

## Adding New Exclusions

1. Document the path and rationale in this file first.
2. Classify as DATA or IDENTITY per the Leverage Principle.
3. Only DATA exclusions are permitted — IDENTITY hits must be fixed.
4. Add the corresponding `--glob` line to `scripts/sovereignty-grep.sh`.
5. Commit both files together.
