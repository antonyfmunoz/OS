# EOS Canonical Template Library
Version: 1.0
Maintained by: Empyrean Creative (EOS developer)

## Governance
Templates updated ONLY by EOS developer after deliberate review.
User instances are built FROM these templates and improve independently.
Canonical layer is invisible to founders.

## Structure
agents/    — Soul doc templates
ventures/  — Venture config templates
skills/    — Skill directory templates
tools/     — Tool integration templates
workflows/ — Workflow templates
standards/ — Operational standards templates

## Instance vs Canonical
Canonical: world-class starting point, built from best practices research,
  shared across all EOS instances.
Instance: instantiated per founder, improves from their outcomes,
  never modifies canonical.
